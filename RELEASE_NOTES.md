# Ergo v1.0.0 — Initial Release 🧠

**Learn anything, anywhere — no internet required.**

## ✨ Features

- **Quiz Engine** — Multiple-choice quizzes with 3 difficulty modes (Plot Armor, Almost Him, Canon Event) and 3 speed modes (Snail, Crunch Time, Panic)
- **Offline-First** — All data stored locally in SQLite; internet only needed for DLC downloads
- **DLC Store** — Browse and download quiz packs from a growing community catalog
- **Analytics Dashboard** — Activity heatmap, performance trends, category breakdowns, behavioral insights (fatigue, tilt, clutch, chronotype), and streak tracking
- **Audio Feedback** — Sound effects for correct/incorrect answers, streaks, and level-ups
- **Onboarding** — Interactive first-launch walkthrough
- **Dark Mode** — Follows system theme automatically
